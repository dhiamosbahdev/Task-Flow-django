# TaskFlow

TaskFlow est une application web de gestion des tâches développée avec Django.

## Objectif du projet

Ce projet pédagogique vise à apprendre Django en construisant une application réelle étape par étape.

## Phases du projet

Phase 1 : application CRUD basique  
Phase 2 : interaction dynamique avec AJAX  
Phase 3 : API REST avec Django REST Framework

## Technologies utilisées

- Django
- HTML
- CSS
- JavaScript
- Git
- GitHub

## Fonctionnalités actuelles

- création d'une tâche
- affichage de la liste des tâches
- gestion via l'interface admin
